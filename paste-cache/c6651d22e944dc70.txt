## 2. Color System

### Primary Palette (Parent Brand)

**Primary Color:**

- **Purpose:** Main brand identifier, CTAs, key elements
- **Considerations:** Must work on light/dark backgrounds, convey trust + innovation
- **Suggested Direction:** Deep blue or tech-forward purple (AI industry standard)

**Secondary Color:**

- **Purpose:** Accents, highlights, interactive elements
- **Considerations:** Complements primary, adds energy

**Neutral Palette:**

- Background grays (light mode)
- Dark mode backgrounds
- Text colors (primary/secondary/tertiary)

### Tier-Specific Colors

**Free Tier Accent:**

- Bright, approachable color (suggest: vibrant blue or green)
- Used for free tier badges, labels, CTAs
- Should feel generous and open

**Premium Tier Accent:**

- Rich, sophisticated color (suggest: gold/amber or deep purple)
- Used for premium badges, exclusive content markers
- Should feel valuable and aspirational

**Deliverables:**

- [ ]  Color palette with HEX/RGB values
- [ ]  Accessibility compliance (WCAG AA minimum)
- [ ]  Usage guidelines (when to use each color)
- [ ]  Gradient options (if applicable)

---

## 3. Typography System

### Font Families

**Headings:**

- Modern, bold, confident
- Good readability at large sizes
- Considerations: Sans-serif, tech-friendly

**Body Text:**

- Highly readable for long-form content
- Works in Skool platform, email, PDFs
- Fallback web-safe fonts

**Code/Technical:**

- Monospace for prompts, code snippets, technical content
- Clear distinction from body text

**Deliverables:**

- [ ]  Font selections with licensing (Google Fonts preferred for web)
- [ ]  Type scale (H1-H6, body, small text sizes)
- [ ]  Line heights and spacing
- [ ]  Weight variations (light, regular, bold)
